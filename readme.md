# Installation
## Prepare
```sh
  # Instalação de dependências
  $ composer install

  # Configure .env e gere a chave
  $ cp .env.example .env
  $ php artisan key:generate
```

## Compile assets
```sh
  # Instalar packages
  $ npm install

  # Rodar tarefas
  $ npm run dev

  
```
## Rode 
```sh
  $ php artisan serve

  # Go to 127.0.0.1:8000
```
